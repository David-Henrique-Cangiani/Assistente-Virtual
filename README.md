# Assistente Virtual por Comando de Voz

Assistente virtual em Python que **executa tarefas por comando de voz**, combinando reconhecimento de fala e automação. Um projeto de **IA aplicada à interação por voz**.

## O problema que resolve

Permite operar tarefas sem o teclado, por voz — útil para acessibilidade, produtividade e automação de comandos do dia a dia.

## Funcionalidades

- Reconhecimento de comandos por voz
- Execução de tarefas automatizadas
- Resposta por áudio ao usuário

## Tecnologias

- Python 3
- Bibliotecas de reconhecimento e síntese de voz (speech recognition / text-to-speech)

## Como rodar

```bash
git clone https://github.com/David-Henrique-Cangiani/Assistente-Virtual.git
cd Assistente-Virtual

# Instale as dependências necessárias (ver código)
pip install -r requirements.txt   # caso exista

# Execute o assistente
python "Assistente Virtual Via Comando De Voz"
```

> Pode ser necessário um microfone configurado e conexão com internet para alguns recursos de voz.
